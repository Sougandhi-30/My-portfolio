LDR and Laser Based Security System

This project uses a laser module and an LDR (Light Dependent Resistor) to create a secure intrusion detection system. The laser module emits a beam of light that is detected by the LDR. When an intruder breaks the laser beam, the LDR detects the change in light intensity and sends a signal to the Arduino board. The Arduino then triggers an alarm and/or sends a notification to the user.
